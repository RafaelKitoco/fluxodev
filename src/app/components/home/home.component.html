O que vai aprender


Poque aprende o Angular?

Angular é um framework de desenvolvimento web open-source mantido pela Google e uma comunidade de desenvolvedores ativos. Ele é amplamente utilizado para construir aplicações web robustas, escaláveis e interativas. O Angular é escrito em TypeScript, uma versão superset do JavaScript, que adiciona recursos de tipagem estática e recursos adicionais para melhorar o desenvolvimento e a manutenção de código

Poque aprende o Uikit?

UIKit é uma biblioteca de interface de usuário (UI) open-source que oferece uma ampla variedade de componentes e estilos pré-construídos para o desenvolvimento web. Ela foi desenvolvida pela equipe da YOOtheme, uma empresa especializada em temas e extensões para sites, e tem como objetivo facilitar a criação de interfaces modernas, atraentes e responsivas.

Como intergrar o uikit em angular
